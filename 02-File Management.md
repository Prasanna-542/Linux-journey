File Management Practice

## Scenario
Organizing project documentation in Linux.

Environment
	•	Practiced on AWS EC2 Linux instance

Commands Practiced
	•	mkdir – create directory
	•	cd – change directory
	•	touch – create file
	•	mv – move/rename files
	•	cp – copy files
	•	ls – list files
	•	tree – display folder structure
